# pawfully-helpful-site

Pawfully Helpful is a multi-page pet care website that provides helpful guides and resources for dog, cat, rabbit, and bird owners.

The goal of this project is to demonstrate front-end development fundamentals including semantic HTML structure, responsive CSS design, accessibility best practices, and basic SEO implementation.

-------------

## Features

- Multi-page website (Home, Dogs, Cats, Rabbits, Birds)
- Responsive layout using modern CSS
- Clean and consistent navigation
- Accessible semantic HTML structure
- SEO-friendly page titles and meta descriptions
- Interactive JavaScript elements
- Organized file and folder structure

-------------

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Vite (build tool)
- Tailwind CSS (plugin configured)
